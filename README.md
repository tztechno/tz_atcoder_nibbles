nibblesでは入力1つ目の数字を「$」、2つ目の数字を「;$」で受け取れる。

```
+_" "=3;@

+    # インクリメントモード(変数の値を1増やす)
\_   # アンダースコアは変数の名前を表す
"    # 文字列モード(次の文字を変数に代入する)
 "   # スペースを変数に代入する(変数の値は" "となる)
=    # 代入モード(変数に値を代入する)
3    # 変数に3を代入する
;    # プリントモード(変数の値をプリントする)
@    # プログラムの終了
```
